# Job Board

## Overview
A job board web application where employers can post job listings
and users can browse and save jobs.

## Roles
- Employer: can create, edit, and delete job listings
- User: can browse and save jobs

## Core Features
- Authentication (login / signup)
- Job CRUD (create, read, update, delete)
- Job search and filtering
- Authorization (only owners can edit/delete jobs)

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: PostgreSQL
- ORM: Prisma
